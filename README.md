# 오후두시랩 과제 (지원자 박재형)

## Goal

- React로 Facebook 코멘트 기능 구현

<br/>

## 요구사항

- [x] 댓글 입력 및 작성 기능
- [x] 좋아요 / 좋아요 취소 기능
- [x] 답글 달기 기능
- [x] 말풍선 메뉴의 댓글 / 답글 삭제 기능
- [x] 기본 프로필 이미지
- [x] 상태 저장은 LocalStorage: 새로 고침 시 코멘트 그대로 노출

<br/>
